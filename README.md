# ☁️ CloudiX - Hospedagem de Bots Discord

Sistema completo de hospedagem para bots Discord, similar à Discloud.

## 🚀 Funcionalidades

- ✅ Sistema de autenticação (Email, Discord, Google, Apple)
- ✅ Verificação de email
- ✅ Dashboard intuitivo
- ✅ Deploy de bots
- ✅ Sistema de planos
- ✅ Perfil de usuário
- ✅ Interface responsiva
